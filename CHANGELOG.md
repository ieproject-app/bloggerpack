## [1.0.0] — In Development

### Added
- Struktur project Bloggerpack (Layout v3)
- Design system berbasis CSS custom properties
- Dark mode (system preference + manual toggle)
- Responsive 2-column layout (main + sidebar)
- Komponen: Header, Navbar, Post Grid, Post Card
- Komponen: Single Post Page, Sidebar, Footer
- Komponen: Pagination, 404 Page
- SEO: Open Graph, canonical URL, structured data (JSON-LD)
- Lazy loading untuk semua gambar
- Back to top button
- Mobile-first responsive design
- Blogger Theme Designer: custom color via skin variables
- Font: IBM Plex Sans + IBM Plex Mono (Google Fonts)
